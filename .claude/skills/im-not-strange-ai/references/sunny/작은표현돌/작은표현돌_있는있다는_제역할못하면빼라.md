# 작은표현돌: '있는/있다는', 제 역할 못 하면 빼라

## 표현
> '있는', '있다는'이 제 역할을 못 하면 빼라. 존재·소유·상태를 실제로 말할 때만 남긴다.

### 예시
| Before | After |
|--------|-------|
| 말라 있는 상태 | 마른 상태 |
| 덮여 있는 | 덮인 |
| 가능성이 있다는 점 | 가능성 |
| 문제가 있는 상황 | 문제가 생긴 상황 / 문제가 있는 상황(실제 존재를 말할 때) |
| 변화가 나타나고 있다 | 변화가 나타난다 |

## 표현 분석
- **상황(situation)**: 퇴고할 때 '있는/있다는' 패턴을 점검하는 맥락. 상태나 존재를 나타내려고 붙인 말이 실제 역할 없이 명사 앞 완충재가 되는 경우가 많다.
- **효과(effect)**: 역할 없는 '있는/있다는'을 빼면 문장이 간결해지고 서술이 바로 선다. 다만 실제 존재·소유·상태를 말하면 유지한다.
- **표현 방식(method)**: Before/After 열거로 즉시 체감. '있다'를 빼거나 관형형('-ㄴ/은')으로 축약하는 전략을 보여준다.
- **저자(speaker)**: Sunny (20년 교열 전문가)

## 활용 판단
- **내 글에 어울리는가?**: 퇴고 체크리스트 항목. '있는', '있다는'을 검색해서 실제 존재·소유·상태를 말하는지 확인한다.
- **어떤 글에 쓰면 좋을 문장인가?**: 모든 글. 특히 보고서와 설명문에서 명사 앞 완충 표현이 빈번하다.
