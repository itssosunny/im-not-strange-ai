# 작은표현돌: 어색한 '있다' 패턴, 다듬어라

## 표현
> '-관계에 있다', '-에게 있어', '-하는데 있어', '-함에 있어' 같은 패턴은 대부분 더 자연스러운 한국어로 바꿀 수 있다.

### 예시
| Before | After |
|--------|-------|
| 가까운 관계에 있었다 | 가까운 사이였다 |
| 그에게 있어 가족은 | 그에게 가족은 |
| 다루는데 있어 | 다룰 때 |
| 연구함에 있어 | 연구할 때에도 |
| 있음에 틀림없다 | 분명하다 |

## 표현 분석
- **상황(situation)**: 퇴고할 때 '있다'가 포함된 관용적 패턴을 점검하는 맥락. 이들은 일본어 번역투나 공문서체에서 온 표현으로, 한국어에서는 어색하다.
- **효과(effect)**: 딱딱한 관용구를 자연스러운 한국어로 바꾸면 문장이 부드러워지고 읽는 속도가 빨라진다. 격식체 느낌을 줄이고 일상적 문체에 가까워진다.
- **표현 방식(method)**: Before/After 열거로 패턴별 교정법을 즉시 보여준다. 각 패턴마다 대체 표현이 다르므로 외워두면 좋다.
- **저자(speaker)**: Sunny (20년 교열 전문가)

## 활용 판단
- **내 글에 어울리는가?**: 퇴고 체크리스트 항목. '있어', '있었다', '있음에'를 검색해서 더 자연스러운 표현으로 교정한다.
- **어떤 글에 쓰면 좋을 문장인가?**: 모든 글. 특히 학술적 글이나 보고서를 읽기 쉽게 다듬을 때 효과적이다.
